================================================================================
【 Unity 씬(Scene) 셋업 가이드 】
================================================================================

이 가이드를 따라 Unity에서 게임을 실제로 작동시킬 수 있습니다.

================================================================================
【 Step 1: 기본 씬 구조 설정 】
================================================================================

1. Unity Editor 열기
2. 새 씬 만들기 (File → New Scene)
3. 씬 이름을 "MainGame"으로 저장

씬 계층 구조 (Hierarchy):
┌─ Canvas (Root UI)
│  ├─ AppClickScreen Panel
│  ├─ LoginScreen Panel
│  ├─ AlgorithmScreen Panel
│  ├─ ReelsScreen Panel
│  ├─ MiniGameScreen Panel
│  └─ EndingScreen Panel
│
└─ GameManager (GameObject)

================================================================================
【 Step 2: Canvas 생성 】
================================================================================

1. Hierarchy에서 우클릭
2. UI → Panel - TextMeshPro 선택
   → Canvas가 자동으로 생성됨
   → Panel이 Canvas 하위에 생성됨

Canvas 설정:
  • Canvas Scaler
    - UI Scale Mode: "Scale With Screen Size"
    - Reference Resolution: (1080, 1920) - 모바일 해상도

Panel 제거하고 다시 시작:
  1. Canvas 선택
  2. Hierarchy에서 삭제 (Canvas만 남기기)

================================================================================

【 Step 6: 테스트 】
================================================================================

1. Play 버튼 클릭
2. 다음 순서로 동작하는지 확인:

✓ 앱 아이콘 클릭 → 로그인 화면으로 전환
✓ 로그인 버튼 클릭 → 테마 선택 화면으로 전환
✓ 테마 선택 → 릴스 화면으로 전환 (게시물 표시, 배터리 100%)
✓ 게시물 클릭 → 반응 (저장 또는 미니게임)
✓ 미니게임 완료 → 배터리 10% 감소, 릴스로 복귀
✓ 배터리 반복 감소 → 1% 도달 시 엔딩 화면

【 자주 하는 실수 해결법 】
================================================================================

❌ 문제: "UIManager를 찾을 수 없습니다" 에러
✓ 해결: UIManager가 Canvas나 다른 오브젝트에 컴포넌트로 추가되어야 함

❌ 문제: 버튼 클릭이 안 됨
✓ 해결: Canvas에 GraphicRaycaster가 있는지 확인
✓ 해결: Button이 Interactable이 ON인지 확인

❌ 문제: 화면 전환이 안 됨
✓ 해결: UIManager의 각 Screen 참조가 올바르게 할당되었는지 확인

❌ 문제: 텍스트가 보이지 않음
✓ 해결: Text의 Color가 검은색인지 확인 (배경이 흰색이면 흰색은 안 보임)
✓ 해결: TextMeshPro를 사용 중인 경우, TextMeshProUGUI 사용 확인

================================================================================
【 추가 팁 】
================================================================================

1. 프리팹 만들기 (나중에 게시물 프리팹 필요할 때):
   - Inspector에서 게시물 Panel을 Project 폴더로 드래그
   - 프리팹 생성됨

2. 씬 저장:
   - Ctrl+S (또는 File → Save Scene)
   - 파일명: MainGame.unity

3. 레이아웃 리셋 (UI가 이상할 때):
   - RectTransform → Right Click → Copy Component
   - 필요한 곳에 Paste Component As New

4. 모바일 테스트:
   - Game 뷰에서 해상도 설정 (1080x1920)
   - 스마트폰처럼 테스트 가능
